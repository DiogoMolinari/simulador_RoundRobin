Simulador Round Robin 

☕ Descrição
Este projeto é um simulador de escalonamento Round Robin (algoritmo de escalonamento que aloca recursos de forma igualitária e circular (em rodízio) entre processos, servidores ou tarefas)
O objetivo é demonstrar conceitos de escalonamento de processos e auxiliar no aprendizado de sistemas operacionais.

⚙️ Funcionalidades
Simula a execução de processos com tempo de execução definido.
Aplica o algoritmo Round Robin com quantum configurável.
Mostra a fila de pronto, a CPU em execução e os processos terminados.
Permite reiniciar a simulação para testar diferentes cenários.

🎨 Visual e Interação
Layout responsivo, compatível com desktops e dispositivos móveis.
Blocos do gráfico Gantt com cores distintas para cada processo.
Botões de iniciar e reiniciar a simulação.
Feedback visual da CPU, fila de pronto e processos finalizados.

🛠️ Tecnologias
HTML5
CSS3
JavaScript

📚 Conceitos Aplicados
Algoritmos de escalonamento de processos (Round Robin)
Manipulação do DOM com JavaScript
Lógica de programação para controle de filas e ciclos de execução
Design responsivo e experiência do usuário (UX)
